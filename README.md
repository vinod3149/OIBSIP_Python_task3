# Encrypted Python Chat Application

This is a secure chat application built with Python. It uses a client-server model and features a graphical user interface (GUI) for easy use. All messages are protected with encryption to ensure privacy.

## ✨ Features

-   **Secure Messaging**: Uses RSA encryption to keep conversations private.
-   **Graphical User Interface (GUI)**: A user-friendly window for chatting.
-   **Real-time Communication**: Messages are sent and received instantly.
-   **Chat History**: Saves conversation history in a local database.

## 🛠️ How to Set Up

**1. Install required libraries:**
Open your terminal or command prompt and run this command:
```bash
pip install -r requirements.txt

```
## 🚀 How to Run
To run the chat application, you'll need to start the server first, and then one or more clients.
**1.Start the server:**
```bash
python server.py
```
**2. Start a Client (in a separate terminal):**
```bash
python client.py    
```
You can open multiple client terminals to simulate a chat between several users.
